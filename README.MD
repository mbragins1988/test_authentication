# Test authentication

Собственная система аутентификации и авторизации. Позволяет пользователям регистрироваться, входить в систему, выходить из учетной записи, обновлять свои данные и удалять аккаунт.

## Авторизация.

Полный доступ к изменению данных пользователей и постов имеет администратор. Модератор может изменять свое имя и фамилию, писать заметки ко всем постам и изменять название и текст своих постов. Пользователь может изменять название и текст своих постов. Просматривать посты могут все, в том числе неаутентифицированные пользователи.
